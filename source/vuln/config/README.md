
# 安全配置错误

## 应用程序栈堆的任何部分都缺少适当的安全加固，或者云服务的权限配置错误

- 高危端口
- 密码配置
    - 默认账户名和密码没有修改
    - 位数过低
    - 字符集小
    - 为常用密码
    - 个人信息相关
        - 手机号
        - 生日
        - 姓名
        - 用户名
    - 使用键盘模式做密码
- 基础配置安全检查
- 安全补丁安装

## 错误处理机制向用户披露堆栈跟踪或其他大量错误信息

- try catch信息输出
- debug信息输出
- stack trace

## 数据库安全配置

- Mongo/Redis等数据库无密码且没有限制访问
- Mysql数据库root账号远程可登陆
